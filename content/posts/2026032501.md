# 新手必看：币安(Binance)交易所C2C买卖币完整操作指南

币安是全球最大的加密货币交易所，拥有超过2.4亿用户。对于许多第一次使用币安的新手小白来说，如何安全、快捷地用人民币购买第一笔数字资产（如USDT）是一个常见的难题。

本文将手把手教你如何通过币安C2C（个人对个人）完成USDT的买卖，使用微信、支付宝即可轻松搞定！

### 一、 账号准备与手续费优惠

在开始交易前，你需要拥有一个币安账户并完成实名认证。

* **新用户注册：** [点击币安交易所的注册链接](https://www.binance.com/join?ref=UKNXKQAK)，使用邀请码 **UKNXKQAK** 进行注册，即可在后续的交易中永久享受 **20%** 的手续费优惠。
* **老用户专属福利：** 如果你之前注册过，但同时满足以下三个条件，参加老用户召回计划，同样可以永久享受 **20%** 的返佣：
    1.  受邀人已经在币安注册账户。
    2.  受邀人账户未绑定过其他推荐关系。
    3.  受邀人账户近180天内未进行过交易。

### 二、 如何入金（C2C买币教程）

如果是小额入金，按照以下步骤操作即可：

1.  **进入C2C交易：** 打开币安App首页，点击**【添加资金】**，选择**【C2C交易】**。
2.  **了解交易区：** 页面上方有三个区域可选：**快捷区**、**自选区**和**大宗区**。大宗区主要针对大额交易，新手小额买币建议直接使用快捷区或自选区。
3.  **输入金额：** 输入你想要购买的人民币金额（例如：1000人民币），系统会自动换算出你大约能收到多少USDT（如139.66 USDT）。
4.  **选择支付方式：** 点击**【选择支付方式】**，你可以根据自己的习惯选择支付宝、微信、银行借记卡等。选择完毕后，点击**【预览订单】**。
5.  **确认商家与下单：** 你有30秒的时间确认订单。在下单前，请务必查看商家的**成交量**和**成交率**（越高越好）。确认无误后点击**【下单】**。
6.  **沟通与转账付款：** * 订单生成后，点击**【聊天】**查看商家要求（如：仅限余额宝支付、需实名付款等）。**必须严格按照商家要求操作以保证交易顺畅。**
    * 获取商家的姓名、收款账号或二维码后，**离开币安平台**，打开你的支付宝/微信/网银向对方转账。
    * 转账完成后，截图保存付款凭证并上传至币安平台，点击**【我已付款，通知卖家】**。商家确认收款后，就会将USDT发放至你的资金账户。

### 三、 如何出金（C2C卖币与防冻卡技巧）

当你需要将USDT变现为人民币时，操作逻辑与买币相反，但**安全审核更为重要**。

1.  **进入自选区卖出：** 回到首页点击**【添加资金】** -> **【C2C交易】**。此时**不建议使用快捷区**，请务必切换到**【自选区】**，并选择**【卖出】**模式。
2.  **设置筛选条件：**
    * 输入你想要卖出的人民币金额。
    * 选择你期望的收款方式（如微信）。
    * **关键一步：** 务必在筛选条件中勾选**【神盾】**。带有“神盾”标志的商家在平台缴纳了大量保证金。如果在交易中或交易后，因商家原因导致你的账户资金被冻结或风控，你可以向平台提交证明并获得一定金额的赔付。
3.  **挑选靠谱商家：** * 关注商家的**成单量**和**成单率**。
    * 点开商家的主页，查看其**账户创建时间**（例如显示已创建722天）。尽量选择注册时间超过一年以上的老商家，账户存活时间越久，通常意味着其资金越安全。
4.  **下单与确认收款：** * 挑选好商家后点击**【卖出】**，输入卖出金额或USDT数量。
    * 添加或选择你的收款账号，点击**【下单】**。
    * **极度重要：** 在收到商家的付款通知后，**绝对不能只看平台的提示！** 务必亲自打开你的银行App、微信或支付宝，核实款项是否**真实到账**。只有在确认实际资金增加后，再回到币安平台点击放币给商家。
