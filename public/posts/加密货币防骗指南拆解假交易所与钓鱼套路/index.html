    一、假交易所骗局的完整套路是什么？ #

    这类骗局有一个非常明显的共同点：无论通过什么渠道联系你，最终都会把你引导到一个假的交易所网站。

    这个假网站做得非常逼真，配色、布局、操作方式几乎和官网一模一样。大多数是网页形式，极少数做了APP。如果你不仔细看网址，真的很难分辨。

    骗局的具体流程： #

    1. 引导访问：骗子以某种理由（例如账号异常公告、KYC认证、平台升级、账号清退等）把你引导到假网站。
    2. 泄露凭证：你在假网站上输入了自己的交易所账号和密码。
    3. 真实登录：假网站后台的骗子拿到你的账号密码，立即去登录真正的交易所。
    4. 触发验证：骗子在真的后台操作，设置「免验证提币地址」，并触发系统给你发送验证码。
    5. 骗取验证码：假网站继续以KYC认证等理由，诱导你把收到的手机验证码和邮箱验证码都输入进去。
    6. 盗取资产：你一旦提交这两个验证码，骗子就成功绑定了免验证地址，之后骗子提取你账户里所有的币，完全不需要任何验证码。

    防线破解原理：正常情况下，提币、转币、卖币都需要输入邮箱验证码加手机验证码（或谷歌验证码）双重验证。但一旦绑定了免验证地址，这道防线就彻底失效了。

    二、收到验证码的时候怎么判断是不是陷阱？ #

    这里有一个非常关键的自救方法，很多人不知道。

    手机短信验证码有时候看不出来是做什么用的，但邮箱验证码会明确告诉你它的用途。当你打开邮箱看到验证码邮件的时候，邮件正文里会清楚写明：

    • 这是一个转账验证码
    • 这是一个提币验证码
    • 这是绑定免验证地址的验证码

    如果你看到邮件里写的是「提币」或「绑定免验证地址」，但你明明没有做这个操作，那就是有人在用你的账号密码登录真正的交易所在操作，这个验证码绝对不能输入到任何地方。

    建议养成习惯：每次收到验证码都先看一眼邮件标题和正文，确认用途再输入。这个习惯花不了5秒钟，但能救你所有的资产。

    四、收到陌生短信说要 KYC 或账号升级怎么办？ #

    骗子通过短信联系你，通常会用这些理由：

    • 升级账号，需要你登录操作
    • 平台 KYC 信息认证
    • 平台限制访问，需要你配合处理
    • 平台迁移账户、账号升级、清退用户
    • 切换 IP 线程、修改账号归属地

    为了取得你的信任，骗子甚至会用繁体字、日语、英语来发短信，利用大多数人看到看不懂的内容会好奇点进去的心理。

    应对法则：不管短信内容看起来多正式，里面的链接一律不要点。如果你担心自己的账号有问题，请直接打开浏览器书签里的官网，或者打开官方 APP，在里面查看账号状态。

    有人在你的微信、Telegram、Facebook等社交媒体上主动联系你，自称是交易所客服或官方工作人员。这种情况其实比假网站更常见。

    这个功能支持验证电话号码、微信、Telegram、Facebook 等主流社交平台，非常实用。

    六、除了假网站和假客服，还有哪些常见骗局？ #

    还有一种骗局越来越常见：以「指导炒币」为名，要求和你进行屏幕共享。

    对方会说带你学习炒币技巧，需要看你的屏幕来指导操作。一旦你允许屏幕共享，对方就能看到你的账号信息、资产情况、验证码，甚至直接操控你的电脑进行转账操作。

    这类情况 100% 是诈骗，不管对方说什么理由都不要答应屏幕共享。

    总结来看，识别骗局有一个最简单的原则：天上不会掉馅饼。 当有人主动找你、给你提供「机会」、让你感觉占了便宜的时候，多问几个懂行的人再操作，不要一个人冲动决定。

    七、常见问题解答 (FAQ) #

    Q：我已经在假网站上输入了账号密码，但还没有输入验证码，现在怎么办？

    A：立刻打开真正的交易所官网或 APP（确认网址无误），马上修改你的登录密码，同时检查账号的绑定设备列表和授权应用，撤销所有陌生设备的访问权限。如果发现有未授权的免验证地址，立即删除。动作要快，骗子拿到账号密码会立刻行动。

    Q：交易所官方会主动联系用户吗？

    A：官方客服不会主动通过微信、Telegram 等社交平台联系你要求你操作账号或提供验证码。如果你需要客服支持，应该自己主动去 APP 或官网的客服入口发起对话。收到主动联系你的「客服」，第一反应就是用「官方验证渠道」查一下真假。

    Q：如果我不确定一条短信或消息的真假，点进去看看链接会怎样？

    A：最好不要点。有些钓鱼链接只是打开就可能触发设备漏洞，或者记录你的 IP 和设备信息。即便什么都没输入，也给骗子提供了线索。正确做法是不点链接，直接用官方验证渠道查号码真假，或者直接联系官方在线客服确认。
