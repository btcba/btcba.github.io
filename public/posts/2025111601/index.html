    二、实名认证（KYC）与安全设置：保护资金最关键的步骤 #

    注册完成后，请立即完成以下安全流程。

    （1）实名认证/KYC #

    路径： 首页 → 头像 → 身份认证

    按系统提示上传：

    • 身份证/护照/驾驶证
    • 完成人脸识别

    KYC一般 3 分钟内即可通过。

    （2）开启双重验证 2FA（必做） #

    路径： 安全设置 → 两步验证（2FA）

    建议同时开启：

    • Google Authenticator（谷歌验证器）
    • 手机短信
    • 邮箱验证码

    双验证是保护资产最重要的防线。

    （3）设置反钓鱼码 #

    设置一个你自己独有的“标识”，所有币安发给你的邮件都会显示该标识，避免钓鱼诈骗。

    （4）开启提现白名单 #

    建议绑定你的常用 USDT 钱包地址，并开启“只允许白名单提现”。

    即使账号被盗，也无法把钱转走。

    三、如何在币安买币（买USDT）｜最安全稳妥的新手方法 #

    国内用户一般通过 C2C（也叫P2P） 使用银行卡/支付宝/微信买USDT。

    步骤如下： #

    ① 进入 C2C 交易 #

    底部导航 →【买币】→【P2P交易】

    ② 选择买入 USDT #

    支付方式可选：

    • 支付宝
    • 微信支付
    • 银行卡

    ③ 选择商家 #

    建议选择：

    • 成交量高
    • 评价好
    • 广告在线

    ④ 输入购买金额并付款 #

    例如：购买 500 RMB → 获得约 70+ USDT （金额会随行情浮动）

    按提示付款 → 上传凭证。

    ⑤ 商家放币 → USDT到账 #

    一般 1 分钟内完成。

    四、币安现货基本使用（买币/卖币/挂单） #

    1. 进入交易页面 #

    【交易】→【现货】

    2. 选择交易对 #

    常用如：

    • BTC/USDT
    • ETH/USDT
    • SOL/USDT
    • BNB/USDT

    3. 买入方式 #

    币安现货提供：

    • 限价单（推荐新手）
    • 市价单（快速成交）

    4. 卖出方式 #

    同样选择限价或市价，填写数量即可。

    新手建议尝试小额，熟悉流程后再正常交易。

    五、币安合约开通流程（USDT合约 / 永续合约） #

    币安合约非常强大，支持做多、做空、杠杆，但同时也伴随更高风险。

    如果你是新手，请一定控制仓位。

    1. 进入合约页面 #

    底部菜单 →【合约】

    2. 完成合约测验 #

    首次开通会出现：

    • 风险测试
    • 使用须知提示

    全部完成后即可使用合约功能。

    3. 合约账户划转 #

    合约需使用合约账户 USDT。

    路径： 【钱包】→【资金】→【转账】 将 USDT 从“现货账户”转至“合约账户”。

    4. 设置杠杆倍数 #

    一般建议：

    • 新手：1–3倍以内
    • 有经验：5倍以内 不要使用 50X、100X 这种极高倍数，非常容易爆仓。

    5. 开多 / 开空 #

    • 开多（Long）：认为价格会上涨
    • 开空（Short）：认为价格会下跌

    设置数量 → 点击下单即可。

    六、如何卖币（C2C卖USDT）｜提现回人民币 #

    当你想把币卖掉换回人民币，可通过 C2C 交易。

    步骤如下：

    ① 打开【买币】-【P2P交易】 #

    切换到 “出售” 页面。

    ② 选择出售 USDT #

    付款方式选择：

    • 支付宝
    • 微信
    • 银行卡

    ③ 选择可靠商家 #

    同样优先选择评价高、成交量大的商家。

    ④ 输入卖出数量 #

    例如卖 100 USDT → 收到人民币 700+（按汇率浮动）

    ⑤ 商家打款 → 确认收款放币 #

    务必等待资金真正到账后再点击“放币”。

    七、出金教程（卖币提现回国内/银行） #

    币安官方不支持直接向银行账户“提现人民币”，但可以通过 C2C/P2P 卖USDT → 接受人民币付款 完成出金。

    流程与【卖USDT】完全一致：

    1. 打开 P2P
    2. 选择出售 USDT
    3. 选择付款方式
    4. 收到人民币后点击放币

    安全提示：

    ✔ 一定要等对方实打实到账 ✔ 不要使用被冻结风险高的账户 ✔ 出金建议分多笔、小额进行

    八、新手安全注意事项（非常重要） #
