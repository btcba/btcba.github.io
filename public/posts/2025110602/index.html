    一、如何进入币安合约交易界面 #

    1. 打开币安APP。

    2. 点击底部导航栏中的【合约】进入合约交易页面。

    3. 并可以看到两种主要合约类型：

      • U本位合约：以 USDT 结算。
      • 币本位合约：用加密货币结算（如BTC、ETH）。

    4. 对新手来说，建议使用更常见、风险相对可控的 U本位合约。

    二、选择交易对（以 ETH 为例） #

    1. 点击左上角币种选择区域。
    2. 在搜索栏输入 ETH。
    3. 选择 ETHUSDT 永续，即可交易 ETH/USDT 的永续合约。

    三、设置保证金模式 & 杠杆倍数 #

    • 在订单右侧，你会看到以下重要设置：

    1. 保证金模式 #

    • 全仓模式：账户内所有资金为合约共同承担风险，一旦爆仓则可能亏光所有保证金。

    • 逐仓模式：每个仓位独立计算，即便爆仓也只损失该仓位的保证金。

      • ✅ 推荐新手使用逐仓模式，风险更透明，便于控制。

    2. 杠杆倍数 #

    • 杠杆倍数支持 1~150倍不等。
    • ⛔ 新手建议选择 3~5倍低杠杆，有效降低爆仓风险。
    • 点击当前杠杆倍数（如20X），选择你的倍数，点【确认】即可。

    3. 保证金模式 #

    这里分为：

    • 单币种保证金模式
    • 联合保证金模式

    一般默认的“单币种保证金模式”即可满足需求。

    四、划转资金到合约账户 #

    1. 下方如出现【可用：0】，说明你的合约账户余额为空。
    2. 点击右侧🔁图标，进入“划转”界面。
    3. 选择从“现货账户➡️U本位合约账户”划转资金。
    4. 填入数量（可点【最大】），点击【确认划转】。

    五、选择订单类型并开仓 #

    1. 订单类型 #

    • 限价单：指定你想要的买入/卖出价格。
    • 市价单：按市场当前价格立即成交，更适合新手和快速操作场景。

    示例：

    如果 ETH 当前价格是 3928 USDT，你可以设定“3920 USDT”作为限价单的买入点，等价格到位即可自动开仓。

    2. 做多或做空？ #

    • 做多：认为ETH价格会上涨，开仓后涨了就盈利。
    • 做空：认为ETH价格会下跌，开仓后跌了就赚钱。

    🚦 小技巧：点击右上角K线图标📈，可以查看行情走势，辅助判断当前是涨还是跌的趋势。

    开仓步骤示例（市价做多 ETH）： #
    1. 选择【市价】。
    2. 拉动数量滑条设置建仓规模。
    3. 点击【买入做多】，并【确认】。

    六、查看持仓与止盈止损设置 #

    一旦成功开仓，你可以在页面下方看到你的：

    • 持仓数量
    • 未实现盈亏（浮亏/浮盈）
    • 开仓价格
    • 强平价格（爆仓价）
    • 保证金比率等信息

    如果你要设置止盈或止损：

    1. 点击持仓区域右侧的【止盈止损】。
    2. 设置你的止盈价格（如3940）和止损价格（如3910）。
    3. 点击【确认】。

    七、平仓与离场 #

    当你觉得已赚够或想及时止损时：

    • 点击右侧【平仓】按钮，并确认即可。
    • 平仓会立即结算该笔合约交易，收益将返还至合约账户。

    总结 #

    无论你是看涨 ETH 做多，还是看跌 ETH 做空，都要牢记合约交易存在高风险，尤其是高杠杆模式下爆仓速度极快。因此：

    ✅ 建议：

    • 控制杠杆倍数
    • 设置止盈止损
    • 保持独立判断，严禁梭哈
