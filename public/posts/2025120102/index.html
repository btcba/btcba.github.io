    👉 官方注册链接（带邀请码）： https://www.binance.com/zh-CN/join?ref=UKNXKQAK 填写邀请码：UKNXKQAK（自动填写）

    ② 点击右上角“三条杠”菜单 #

    进入主页后，在右上角会看到“三”图标，点开侧边栏菜单。

    ③ 往下滑 → 找到【下载】 #

    进入菜单后拉到最底部，会看到一个【下载】按钮，点击进入下载页面。

    ④ 点击【安卓下载】 #

    根据你的手机系统选择，这里我们点击 Android 下载。

    ⑤ 点击【下载】正式开始 #

    系统会弹出“是否保存文件”，点击 下载。 看到下载文件出现后，点进去即可。

    ⑥ 文件正在下载 → 下载完成点击打开 #

    等待几秒钟，下载完成后点 打开 安装包。

    ⑦ 安装前授权：选择“允许本次安装” #

    首次安装非应用商店软件，安卓会提示：

    是否允许来自此来源的安装？

    点击 允许。

    ⑧ 出现安全增强防护 → 点击“开始深度扫描” #

    部分手机（如华为、小米、OPPO）会自动启动安全检测，属于正常情况。

    ⑨ 扫描提示为风险软件 → 点击【查看详情】 → 【仍然安装】 #

    币安是全球最大交易所之一，被标记只是因为未从应用商店安装，属于正常现象。 直接点击：

    • 查看详情
    • 仍然安装

    即可继续完成安装流程。

    ⑩ 安装完成 → 币安APP成功下载 #

    安装结束后，点击 打开，即可进入币安主界面。 至此，安卓下载流程完成！

    二、币安账号注册完整教学（大陆用户专用流程） #

    下面继续教你如何注册账户 & 完成KYC实名认证。

    ① 打开币安APP → 点击「注册」 #

    选择通过 邮箱 或 手机号码 注册。 建议使用邮箱更稳妥，不受短信屏蔽影响。

    ② 填写注册信息 #

    • 输入邮箱
    • 设置登录密码（建议8位以上，包含大小写+数字）
    • 确认邀请码：UKNXKQAK（自动带入）

    点下一步。

    ③ 输入邮箱验证码 #

    币安会发验证码到你的邮箱，把验证码填进去即可。

    ④ 设置两步验证（2FA） #

    为了账户安全，建议开启：

    • 邮箱验证码
    • 谷歌验证器（推荐）
    • 手机验证（如能正常接收也可）

    开启2FA后账户安全性大幅提升。

    ⑤ 开始 KYC 身份认证 #

    要开启充值、买币、提现功能，都需要通过实名认证。 点击「用户中心」→「身份认证」。

    认证步骤如下： #

    1. 选择国籍：中国大陆

    2. 上传证件（建议使用护照，成功率高）

    3. 系统要求拍摄：

      • 证件正面
      • 证件信息页
      • 本人自拍&动态识别（按提示左右转头）

    整个过程不到2分钟。

    ⑥ KYC审核完成 → 即可买币交易 #

    实名认证一般几分钟即可通过，通过后你就可以：

    • 进行P2P买币（C2C）
    • 交易USDT、BTC
    • 使用币安合约
    • 提币转账到外部钱包

    至此，币安App注册流程全部完成！
