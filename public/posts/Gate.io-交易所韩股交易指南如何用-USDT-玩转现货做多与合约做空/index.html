    近年来，韩国 AI 半导体板块表现极其亮眼。SK海力士、三星电子等行业巨头的股价波动与国际化进程，吸引了全球投资者的目光。然而，传统的韩股投资渠道门槛极高，非韩国本土居民极难直接参与。

    目前，Gate.io 交易所已支持通过 USDT 直接交易超过 1,000 只韩股，无需兑换韩元，极大地降低了参与门槛。本文将为你系统梳理韩股的基础交易规则，并详细演示在 Gate.io 进行现货买入（做多）与合约交易（做空/做多）的完整实操流程。

    一、 交易前必知：韩股核心市场规则 #

    在正式入场前，必须厘清韩股市场的运行规则，这与 A 股或美股存在较大差异：

    1. 板块划分：
    • 主板：以三星、现代、SK海力士、LG 等巨型财阀企业为主。其中，SK海力士和三星电子两家巨头占主板指数权重高达 40%，具有决定性的风向标作用。
    • 创业板：以科技、互联网、生物医药以及文娱企业（如 HYBE、SM 娱乐等）为主，弹性更大。

    1. 交易时间：北京时间 08:00 - 14:30（中午不休市）。Gate.io 虽然支持 7×24 小时交易，但非非盘交易时间（如夜盘、周末）流动性较差、滑点较大，强烈建议选择正常开盘时段进行交易。
    2. 涨跌幅限制：单日涨跌幅限制为 ±30%，波动显著高于 A 股。
    3. 熔断机制（Circuit Breakers）：当大盘指数较前一日收盘价分别下跌 8%、15%、20% 时，将触发暂停交易。
    4. 极低门槛（碎股交易）：最低支持 0.01 股起购，极大降低了散户的资金门槛。
    5. 汇率敞口：韩股虽以韩元计价，但 Gate.io 已自动换算。需要注意的是，韩元对主流货币的汇率波动也会间接影响资产的实际价值。

    二、 现货做多：如何用 USDT 直接买入韩股？ #

    如果你看好某只韩股，希望长期持有或做多，最安全、直接的方式是购买现货。

    1. 寻找韩股入口 #

    打开 Gate.io App，在首页点击 「股票」 板块，进入后切换至 「韩股」 分类。页面会优先展示三星电子、SK海力士等热门标的，你也可以通过搜索栏直接检索目标个股。

    2. 现货买入步骤（以 SK 海力士为例） #

    1. 进入交易页：点击 SK海力士 进入详情页，点击下方的 「买入」。
    2. 选择订单类型：
    • 限价单：设定你期望的特定价格，等待市场撮合。
    • 市价单：以当前市场最优价格立即成交（推荐在开盘期间使用，成交更迅速）。

    1. 输入数量：支持碎股交易，最低可输入 0.01 股。
    2. 确认交易：确保现货账户中留有足够的 USDT，点击 「买入」 并确认。

    3. 持仓与卖出 #

    交易完成后，在页面下方即可实时查看持仓明细和浮动盈亏。若需止盈或止损离场，直接点击 「市价全平」 并确认即可。

    • 注：实测现货交易手续费约为订单金额的 0.1%（千分之一）左右，具体以平台实时结算为准。

    三、 合约交易：如何利用杠杆做空或做多韩股？ #

    由于真实的韩股现货不开放做空，若你判断某只股票即将下跌，或者希望通过杠杆放大收益，可以通过 Gate.io 的永续合约功能来实现。

    1. 划转资金至合约账户 #

    在操作前，需将 USDT 从现货账户划转至合约账户：

    1. 切换至 「合约」 界面。
    2. 在“可用余额”旁点击 「+」（划转）。
    3. 选择从 「现货账户」 划转至 「USDT永续账户」，输入金额并确认。

    2. 检索合约标的 #

    在合约交易页面的搜索框内，输入想要交易的股票名称（支持中文或英文，如输入“三星电子”或“SAMSUNG”），选择对应的 「SAMSUNGUSDT 永续」 合约。

    3. 开仓关键设置（至关重要） #

    为了控制风险，在点击开仓前，请务必完成以下两项设置：

    • 将「全仓」改为「逐仓」：

    • 全仓模式下，若仓位面临爆仓，系统会动用你合约账户中的所有余额进行补仓，风险极高。

    • 逐仓模式下，单笔交易的亏损上限仅限于该笔仓位分配的保证金，不会波及账户内的其他资金。

    • 合理设定杠杆倍数： Gate.io 虽然对部分个股合约提供高达 50 倍的杠杆，但韩股本身波动剧烈，高杠杆极易导致瞬时爆仓。建议将杠杆控制在 3x - 5x 之间。

    4. 实施做空（或做多）与设置止盈止损 #

    以做空三星电子为例：

    1. 选择委托类型：选择「市价」以快速锁价成交，或选择「限价」埋单。
    2. 输入仓位大小：拖动百分比滑块或手动输入数量。
    3. 启用止盈止损（必须养成习惯）：
    • 止盈价格：做空时，价格下跌盈利。设定一个低于当前开仓价的目标价（如当前 185，设为止盈 180）。
    • 止损价格：做空时，价格上涨亏损。设定一个你所能承受的最大反向波动价格（如设为止损 190）。
    1. 方向选择：
    • 判定价格下跌：点击 「卖出做空（开空）」。
    • 判定价格上涨：点击 「买入做多（开多）」。

    成交后，在底部的“持有仓位”中即可实时监控强平价格、保证金比例及未实现盈亏。需要手动离场时，点击 「平仓」 即可。

    四、 常见问题解答 (FAQ) #

    Q：Gate.io 的韩股交易和直接在韩国券商炒股有什么区别？ A： Gate.io 的韩股现货锚定的是真实的韩股价格走势，并支持 100% 准备金，其最大优势在于免去了繁琐的法币结汇（韩元）流程，直接使用 USDT 结算。而合约则是纯粹的衍生品，不涉及实物股票，仅用于价格波动的多空博弈。

    Q：为什么不建议在非交易时间（夜间/周末）操作？ A： 尽管平台支持 7×24 小时挂单，但在韩国交易所休市期间，平台内的挂单量会急剧减少。此时交易会导致买卖价差（Spread）异常拉大，滑点严重，容易以极其不划算的价格成交。

    Q：如果逐仓合约面临爆仓，我可以追加保证金吗？ A： 可以。在逐仓模式下，你可以手动为该笔特定仓位增加保证金以降低强平价格（降低风险）。但在波动极快的市场中，最稳妥的防爆手段依然是严格设置止损单，而非盲目抗单追加保证金。
