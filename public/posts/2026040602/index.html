    • 邮箱：建议 Gmail、Outlook 等国际邮箱，避免国内邮箱收不到验证邮件；
    • 手机号：最好是境外手机号，如果没有可用邮箱注册后再绑定；

    二、注册流程 #

    1. 打开币安官网或下载 App；
    2. 点击 注册，选择邮箱或手机号；
    3. 填写邀请码 BTC147；
    4. 设置登录密码（建议包含大小写+数字+符号）；
    5. 完成邮箱/手机验证码验证。

    三、身份认证（KYC） #

    为了保证账户安全并提高交易额度，需要完成 KYC：

    1. 登录币安后，进入**【身份认证】**；
    2. 上传身份证/护照正反面；
    3. 进行人脸识别；
    4. 等待审核（一般几分钟到几小时完成）。

    四、安全设置 #

    账户安全非常关键，建议完成以下操作：

    • 开启 Google 身份验证器（2FA）；
    • 绑定手机号或邮箱双重验证；
    • 设置资金密码与防钓鱼码；
    • 开启提现白名单功能，防止盗提。

    五、买币操作（以人民币购买 USDT 为例） #

    1. 登录币安 → 进入**【交易】** → 【P2P】；
    2. 选择 USDT，支付方式选 银行卡 / 支付宝 / 微信；
    3. 选择信誉良好的商家，下单购买；
    4. 支付成功后，USDT 将到账币安账户。（后续可用 USDT 去交易比特币、以太坊等主流加密货币）

    六、防冻卡技巧（大陆用户必看） #

    • 备注栏不要填写“USDT”、“比特币”、“Crypto”等敏感字样；

    • 建议多准备一张收付款银行卡备用。
