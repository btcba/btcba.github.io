    一、币安（Binance）平台简介 #

    币安由加拿大华裔工程师赵长鹏于 2017 年创立，已迅速成长为全球排名前列的数字货币交易平台。它不仅提供友好的交易界面，支持多语言和广泛的币种，更构建了一个完整的数字资产生态系统，包括币安智能链（BSC）、币安币（BNB）和币安NFT等。

    核心优势： 交易量巨大（日交易额可达 $2.0 亿美元）、高吞吐量（每秒处理超过 1,400,000 笔交易）、以及 7*24 小时客户服务。

    中国人可以使用币安吗？

    可以。 尽管中国大陆对加密货币交易有严格限制，但通过遵守当地法律法规并采用适当的技术手段，中国用户可以访问并使用币安平台。

    由于监管原因，中国大陆用户在注册时需要注意以下关键点，以确保流程顺利：

    • 目的： 绕过网络封锁，顺利访问币安官网。

    • 要求： 准备好中国大陆护照或身份证。实名认证（KYC）时需上传真实、有效的证件。

    注册成功后，必须完成 KYC 身份验证才能进行交易。

    2. 国籍选择： 在 KYC 身份认证环节，国籍请选择“中国”（China）。
    3. 上传证件： 按照系统提示，上传您的中国大陆身份证或护照照片。
    4. 人脸识别： 完成人脸识别验证（按提示做出摇头、张嘴等动作）。

    认证结果： 使用非中国大陆 IP + 中国国籍证件完成认证，即可开通账户的全部功能。

    四、资金出入金（C2C 交易）和使用指南 #

    币安为用户提供便捷的法币（人民币）出入金通道，即 C2C (Customer to Customer) 交易。

    1. 购币流程（入金） #

    1. 切换法币： 在币安 App 内，将法币显示单位从 USD 切换为 CNY。
    2. 进入 C2C 交易： 在主页选择“C2C 交易”或“快捷买币”功能。
    3. 选择商家/方式： 选择您要购买的币种（通常是 USDT），选择信誉良好的商家，并选择支付方式（支付宝、微信或银行卡）。
    4. 转账： 按照商家提供的账户信息进行转账。切记：转账时不要在任何地方备注“比特币”、“加密货币”、“USDT”等任何敏感信息。
    5. 确认放行： 转账完成后，点击“已付款”，等待商家确认收款并将数字货币（如 USDT）转入您的币安账户。
    6. 开始交易： 获得 USDT 后，即可使用“币币交易”功能购买比特币（BTC）、以太坊（ETH）等其他币种。
