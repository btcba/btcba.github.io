    ⚠️ 注册前的核心注意事项 #

    由于当前的监管政策与网络环境，中国大陆用户在注册和使用币安时，请务必注意以下两点，以确保流程顺利：

    1. 网络环境 (IP地址)： 全程必须使用科学上网工具。
      • 避开地区： 美国、新加坡、中国大陆、加拿大。
      • 推荐节点： 台湾、韩国、日本或欧洲地区节点。
    2. 身份认证 (KYC)： 目前币安已支持中国大陆用户使用身份证或护照完成实名认证（无需必须使用护照，身份证通过率同样很高）。

    🛠️ 第一步：前期准备工作 #

    在开始注册前，请准备好以下三样东西：

    1. 电子邮箱： 推荐使用 Outlook, Gmail, iCloud 等国际邮箱。国内邮箱（如QQ、163）可能存在收不到验证码的风险。
    2. 证件： 中国大陆身份证或护照（需在有效期内）。
    3. 币安 App 下载（重要）：
      • 安卓用户： 请直接去币安官网下载 APK 安装包（不要去应用商店）。
      • iOS 用户： 需要使用非中国大陆地区的 Apple ID（如美区、台区 ID）在 App Store 搜索 “Binance” 下载。

    📝 第二步：币安注册详细流程 #

    1. 进入注册页面 #

    点击币安官方注册链接（已内置 20% 手续费折扣）： 👉 点击此处注册币安 (Binance) 或者在注册时手动填写推荐码：UKNXKQAK

    2. 填写账户信息 #

    • 选择 “通过邮箱/手机号注册”。
    • 输入您的 电子邮箱（推荐）或手机号（如使用+86手机号，请确保网络节点稳定）。
    • 设置一个强密码：必须包含 8-20 个字符，至少 1 个大写字母、1 个数字。
    • 勾选同意服务条款，点击“下一步”。

    3. 输入验证码 #

    系统会向您的邮箱发送 6 位数验证码，输入后即可完成账号的初步创建。

    🆔 第三步：身份认证 (KYC) #

    注册完成后，必须完成 KYC 才能进行交易。建议在手机 App 端进行认证，操作更流畅。

    1. 登录 App： 使用刚注册的账号登录币安 App。
    2. 选择地区： 系统通常会自动识别，如果没有，请手动选择 “China (中国)”。
    3. 上传证件：
      • 选择证件类型（身份证或护照）。
      • 拍摄证件的正反面（确保光线清晰，无反光，四角完整）。
    4. 人脸识别： 按照屏幕提示完成点头、张嘴或眨眼等动作。
    5. 等待审核： 通常在 10 分钟至 1 小时内审核通过，审核通过后您将获得“已认证”标识。

    🛡️ 第四步：账户安全设置（必做） #

    为了资金安全，请务必开启 双重身份验证 (2FA)：

    1. 在 App 首页点击左上角图标 -> 安全。
    2. 通行密钥/生物识别： 开启指纹或面容ID登录。
    3. 身份验证器 App： 推荐下载 Binance Authenticator 或 Google Authenticator，绑定后提现和登录都需要动态验证码，极大提升安全性。

    💰 第五步：如何在币安购买加密货币 (C2C 交易) #

    对于新手，最常用的方式是通过 C2C (C2C Trading) 使用人民币购买 USDT。

    1. 切换货币单位 #

    打开 App，点击左上角头像 -> 设置 -> 汇率/货币，确保选择为 CNY (人民币)。

    2. 进入 C2C 交易区 #

    • 在首页点击 “充值” 或 “C2C 交易”。
    • 选择 “快捷区” (系统自动匹配) 或 “自选区” (自己挑选商家)。

    3. 购买 USDT #

    1. 选择 “购买” -> “USDT”。
    2. 输入您想购买的金额（例如 1000 CNY）。
    3. 选择支付方式（支付宝、微信或银行卡）。
    4. 下单与付款（关键）：
      • 下单后，您会看到卖家的收款信息。
      • 请离开币安 App，去您的支付宝/银行卡 App 转账给卖家。
      • ❌ 严禁备注： 转账时千万不要备注任何敏感词（如：BTC、USDT、币安、加密货币等），否则会导致银行卡被冻结！什么都不要写，或者仅填写卖家要求的随机码。
    5. 确认付款： 转账成功后，必须回到币安 App 点击 “我已付款，通知卖家”。
    6. 等待放币： 卖家确认收款后，USDT 会自动划转到您的币安 “资金账户” 中。

    恭喜！完成以上步骤后，您就已经成功踏入加密货币的世界了。

    • 交易入口： 拥有 USDT 后，您可以点击底部 “交易” 按钮，用 USDT 购买比特币 (BTC)、以太坊 (ETH) 等任意币种。
    • 风险提示： 币圈波动大，投资需谨慎，请务必保管好您的账号密码及私钥信息。
