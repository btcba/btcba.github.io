    U本位合约和币本位合约有什么区别？ #

    点击底部的「合约」按钮，进入交易界面，顶部会有两个选项：

    • U本位合约： 用 USDT 做保证金，保证金本身价格稳定。你赚的亏的都是 USDT。
    • 币本位合约： 用 BTC 或其他加密货币做保证金。上涨趋势做多时，除了合约收益，你的保证金本身也在涨，等于双倍收益；但下跌时就是双倍亏损。

    新手建议： 必须且只玩 U本位合约，因为保证金固定，没有额外汇率波动，最容易理清盈亏。

    核心参数设置：全仓/逐仓 与 杠杆倍数 #

    1. 全仓和逐仓怎么选？

    • 全仓： 账户有100块，开1块钱仓位。如果爆仓，100块全部亏完。
    • 逐仓： 账户有100块，开1块钱仓位。如果爆仓，只亏这1块钱，剩下99块安全。
    • 新手建议：必须选择逐仓！ 风险绝对可控。

    2. 杠杆倍数怎么设置？ 在仓位模式右侧可以调整倍数（1X - 125X）。新手建议最高只开 10X。 需要注意的是，你设定的 10X 只是“表面杠杆”。实际杠杆 = 仓位金额 ÷ 账户总资金。

    合约实战：开仓、平仓与止损 #

    第一步：划转资金 点击可用余额旁边的划转按钮，把 USDT 从「现货账户」划转到「U本位合约账户」。

    第二步：选择市价单还是限价单？

    • 限价委托： 自己设定一个理想价格，等市场跌到或涨到这个价格才成交。手续费低，但可能会踏空。
    • 市价委托： 按当前市场最新价格瞬间成交。速度最快，但手续费稍贵。

    第三步：开仓做多（或做空）

    1. 选择市价单（演示用）。
    2. 在数量栏输入金额（比如 50 USDT）。
    3. 系统会显示预估强平价格（也就是跌到什么位置你会爆仓）。
    4. 确认无误后，点击「买入/做多」。

    第四步：设置止盈止损（保命神技） 持有仓位后，在下方仓位列表中点击「止盈止损」：

    • 设置止盈价格（赚到多少自动落袋为安）。
    • 设置止损价格（亏到多少必须割肉离场）。 系统会自动帮你盯盘，到达点位自动触发。

    第五步：如何平仓？ 想立刻结束战斗，直接在仓位列表点击「一键平仓」，系统会按市价结算你的所有盈亏，钱重新变回 USDT 回到你的合约账户中。
