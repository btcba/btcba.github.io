    2. 选择居住国家

    3. 邮箱注册

    6. 设置密码
      绑定手机后，为账户设置一个安全密码，完成注册。

    身份认证及安全设置 #

    1. 身份认证
      注册成功后，需要完成身份认证才能使用交易功能。点击【立即认证】。

    2. 选择证件类型
      在认证页面选择“身份证”或“护照”，然后上传身份证件正反面，确保图像清晰且四角完整。

    3. 人脸识别
      上传身份证后，进行简单的人脸识别，完成个人信息确认并提交审核。

    4. 安全设置
      身份认证成功后，访问【安全中心】进行安全设置。建议添加通行密钥、身份验证器 APP、防钓鱼码和紧急联系人，以最大程度保护账户安全。

    下载欧易APP #

    1. 下载二维码
      完成身份认证后，点击右上角的箭头扫描二维码下载欧易 APP（安卓手机）。

    2. 苹果手机下载
      苹果用户需拥有海外 Apple ID，可在谷歌浏览器搜索“台湾 ID”找到卖家购买。

      如果 APP 默认显示英文，点击左上角的点阵，选择个人资料，然后在语言设置中切换为简体中文。

    4. 切换到交易所模式
