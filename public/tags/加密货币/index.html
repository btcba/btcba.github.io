    永续合约，全名永续期货合约（Perpetual Futures Contract），简称 PERP。 例如：BTCUSDT 永续合约，会称为 BTC-PERP 或 BTC 永续杠杆合约。

    它从传统期货演变而来，但最大的差别在于：

    • 没有到期日（你可以无限期持仓）
    • 透过资金费率与标记价格机制，维持合约价格与现货价格的平衡
    • 不需要持有现货，也能做多、做空、开杠杆

    2. 做空（看跌） #

    3. 杠杆倍率（1～125 倍） #

    杠杆让你用小本金放大仓位。

    • 用 100 美元开 10 倍多单 等于操作 1,000 美元仓位 涨 10% → 赚 100 美元 跌 10% → 亏光本金 → 爆仓

    4. 保证金与强平机制 #

    • 初始保证金：开仓所需的抵押（如你投入的 100U）
    • 维持保证金：仓位必须维持的最低资金
    • 当亏损接近本金时 → 系统自动强制平仓（爆仓）
