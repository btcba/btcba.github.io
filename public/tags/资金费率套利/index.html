    进入费率套利界面后，点击「参考年化」表头进行降序排列，把收益率最高的标的排到最上面。

    2. 理解负费率的套利逻辑 #

    假设排名第一的标的当前资金费率是 -0.091%，下期预测费率是 -0.13%。

    • 资金费率为负： 代表市场上做空的人更多，做空方需要向做多方支付利息。
    • 如何吃息： 想要“收钱”，就必须在永续合约里开多。
    • 如何对冲： 永续合约开多后会面临单边下跌的风险，因此必须在现货市场做空来对冲。由于现货不能直接做空，需要使用「现货杠杆」，即借入该代币并卖出。

    三、 参数设置与一键下单（以 CRO 为例） #

    假设选择高收益的 CRO 代币进行套利，具体的“双腿”参数设置如下：

    • 右腿（永续合约）： 因为费率为负，系统会自动设定为买入开多永续合约（例如全仓 3 倍杠杆）。
    • 左腿（现货杠杆）： 选择杠杆账户，搜索 CRO/USDT。最大支持 5 倍杠杆，建议与合约保持一致（如 3 倍杠杆）。操作方向为借入 CRO 后卖出。
    • 价格类型： 强烈建议选择「市价」。如果是限价单，由于行情波动，很容易出现一条腿成交、另一条腿未成交的情况，从而暴露在单边风险中。系统通常会显示 0.28% 到 0.4% 的价差率，这个市价滑点在套利中属于可接受范围。
    • 投入金额： 套利交易通常以 USDT 为本位。勾选「同金额」，填入你的总投入金额（例如 200 USDT），系统会让两条腿各自按 200U 的等值去下单，保持仓位绝对平衡。

    第一步：搞懂两种套利的核心逻辑 #

    在实操前，我们需要先理解我们到底在赚谁的钱。

    1. 资金费率套利（赚利息） #

    永续合约为了让合约价格不偏离现货价格，引入了“资金费率”机制。简单来说：

    • 做多的人太多（费率为正）： 多头每隔 8 小时要付一笔利息给空头。
    • 做空的人太多（费率为负）： 空头每隔 8 小时要付一笔利息给多头。

    套利逻辑： 我们永远站在“收钱”的那一方。比如某币种费率是 0.10618%，我们开 1000 刀的多单，每 8 小时就能白嫖 1000 × 0.19984% = 1.0618 刀。同时，为了防止币价上涨导致空单亏损，我们买入等价值的现货。现货和合约涨跌互抵（风险敞口为零），我们只安心收利息。

    2. 价差套利 / 期限溢价（赚差价） #

    打开交割合约（比如次季度交割），你会发现同一个币，交割时间越远，价格往往比现货越高，这就产生了“溢价”。

    • 举个生活化的例子： 现在现货苹果 1 元/个，但两个月后的苹果期货标价 1.1 元。
    • 套利逻辑： 你现在花 1 元买入现货苹果，同时在期货市场以 1.1 元卖出（做空）。等到期时，不管苹果涨跌，这个 0.1 元的价差你吃定了。这就是赚取“溢价随着时间推移必然收敛到零”的确定性利润。
