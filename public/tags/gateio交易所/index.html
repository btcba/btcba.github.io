    Gate 开通并购买美股完整教程 #

    Gate 目前提供 两种方式 交易美股：股票现货交易（对接合规券商，直接买卖美股股票和 ETF）和 CFD 差价合约（以 USDT 为抵押，带杠杆交易美股差价合约）。两种方式的开通和操作流程不同，下面分别讲解。

    方式一：股票现货交易（推荐新手使用） #

    Gate 股票现货交易通过对接合规券商，提供真实的美股股票与 ETF 交易服务，不是链上映射资产或代币化衍生品，而是合规的真实股票交易。 第 1 步：注册 Gate 账户

    如果你还没有 Gate 账户，先完成注册： 注册链接：https://www.gate.io/signup/U1UXUV8K?ref_type=103 邀请码：U1UXUV8K

    通过此链接注册，邀请码会自动填入。

    第 2 步：完成 KYC 身份认证 #

    登录 Gate App 后： 点击左上角个人头像 → 个人资料 →「身份认证」 选择证件类型（身份证/护照/港澳通行证），填写个人信息并上传证件照片 完成人脸拍照验证

    ⚠️ KYC 认证是开通美股交易的必要前提，未完成认证无法进行后续操作。

    将 Gate App 更新至 V8.21.5 及以上版本，这是股票现货交易的最低版本要求。可在应用商店搜索 Gate 进行更新。

    Gate 注册教程：使用邀请码快速开户指南 #

    一、注册前准备 #

    注册前请准备好以下资料： 一个常用邮箱或手机号（用于账号登录和接收验证码） 身份证件：身份证、护照或驾照（择一，用于后续 KYC 身份认证）

    二、注册步骤 #

    第 1 步：打开注册链接

    直接点击以下邀请链接进入 Gate 注册页面： 注册链接：https://www.gate.io/signup/U1UXUV8K?ref_type=103

    通过此链接注册，邀请码会自动填入，无需手动输入。 第 2 步：选择居住国家/地区

    进入注册页面后，首先选择你的居住国家/地区。这一步很重要，请如实选择，不同地区可用的服务和功能可能略有差异。 第 3 步：填写邀请码

    如果你没有通过邀请链接进入，而是直接访问了 Gate 官网，则需要手动填写邀请码： 邀请码：U1UXUV8K

    操作方式：点击"创建账户"后，在邮箱/手机号输入栏下方，点击"邀请码"右侧的下拉箭头展开输入框，将邀请码填入即可。

    ⚠️ 重要提醒：邀请码只能在注册时填写，注册完成后无法补填。如果忘记填写，需要用新邮箱重新注册一个账号（同一身份最多可认证 5 个账号）。 第 4 步：创建账户

    填写以下信息完成账号创建： 邮箱或手机号 设置登录密码（建议使用强密码，包含大小写字母、数字和特殊字符） 确认邀请码已填入 点击"创建账户" 第 5 步：完成验证

    系统会向你填写的邮箱或手机号发送验证码，输入验证码后即完成账号注册。 第 6 步：KYC 身份认证（建议完成）

    注册成功后，建议尽快完成 KYC 身份认证： 登录 Gate 账号 进入身份认证页面 选择证件类型（身份证/护照/驾照） 填写个人信息并上传证件照片 完成人脸拍照验证

    KYC 认证全程约 3 分钟即可完成，认证后可解锁充值、交易、提现等完整功能，并有机会获得新人奖励。 三、注册后建议操作 开启安全设置：绑定手机号/邮箱二次验证，设置资金密码，提升账户安全性 首次充值：通过银行卡、第三方支付或链上转账向 Gate 账户充值 开始交易：充值完成后即可进行现货或合约交易
