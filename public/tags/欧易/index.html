    什么是合约交易？ #

    合约交易（Futures Trading） 是加密货币市场中一种流行的衍生品交易方式。简单来说，你并不是在买卖真正的比特币或以太坊，而是在交易一份预测未来价格走势的合同。

    通过合约，你可以：

    • 看涨（做多/Long）： 如果你认为价格会上涨，买入合约。
    • 看跌（做空/Short）： 如果你认为价格会下跌，卖出合约。

    合约交易的核心概念 #

    在开始操作前，必须掌握以下几个核心术语：

    1. 杠杆（Leverage） #

    杠杆是合约交易的“放大镜”。例如，你使用 10x 杠杆，只需 100 USDT 就可以操控价值 1000 USDT 的仓位。

    • 优点： 放大收益。
    • 缺点： 同样放大亏损，价格反向波动 10% 就会导致本金归零。

    2. 保证金（Margin） #

    保证金是你开仓时抵押在交易所的资产。

    • 逐仓模式： 亏损仅限于该仓位投入的保证金。
    • 全仓模式： 账户内所有余额都会作为保证金，风险更高但更难被爆仓。

    3. 强平（Liquidation/爆仓） #

    当市场走势与你的方向相反，且亏损达到保证金上限时，系统会强制关闭你的仓位以防止欠款。这就是俗称的“爆仓”。

    4. 交割合约 vs 永续合约 #

    • 交割合约： 有固定的结算日期（如季度合约）。
    • 永续合约： 没有到期日，通过“资金费率”机制让合约价格锚定现货价格，是目前主流的交易品种。
