    第一步：搞懂两种套利的核心逻辑 #

    在实操前，我们需要先理解我们到底在赚谁的钱。

    1. 资金费率套利（赚利息） #

    永续合约为了让合约价格不偏离现货价格，引入了“资金费率”机制。简单来说：

    • 做多的人太多（费率为正）： 多头每隔 8 小时要付一笔利息给空头。
    • 做空的人太多（费率为负）： 空头每隔 8 小时要付一笔利息给多头。

    套利逻辑： 我们永远站在“收钱”的那一方。比如某币种费率是 0.10618%，我们开 1000 刀的多单，每 8 小时就能白嫖 1000 × 0.19984% = 1.0618 刀。同时，为了防止币价上涨导致空单亏损，我们买入等价值的现货。现货和合约涨跌互抵（风险敞口为零），我们只安心收利息。

    2. 价差套利 / 期限溢价（赚差价） #

    打开交割合约（比如次季度交割），你会发现同一个币，交割时间越远，价格往往比现货越高，这就产生了“溢价”。

    • 举个生活化的例子： 现在现货苹果 1 元/个，但两个月后的苹果期货标价 1.1 元。
    • 套利逻辑： 你现在花 1 元买入现货苹果，同时在期货市场以 1.1 元卖出（做空）。等到期时，不管苹果涨跌，这个 0.1 元的价差你吃定了。这就是赚取“溢价随着时间推移必然收敛到零”的确定性利润。

    1. 筛选高收益币种 #

    • 进入“费率套利”后，将计价方式切换为 USDT 本位（逻辑简单，无需持有特定币种作保证金）。
    • 点击「参考年化」列表头，进行降序排列。排在最前面的就是当下收益最可观的币种。

    2. 下单步骤（以费率为正为例） #

    当资金费率为正数时，我们需要：买入现货 + 永续合约做空。

    1. 点击目标币种进入下单界面。
    2. 确认方向：现货选择「买入」，永续合约选择「卖出开空」。
    3. 调整杠杆：建议设为 3 倍 - 5 倍（不要超过 5 倍，确保风险可控）。
    4. 输入金额：填入你想投入的 USDT 数量（例如 1000 USDT）。
    5. 勾选「同金额」：确保现货和合约用同等资金对冲。
    6. 勾选「任意一腿完成成交，另一腿直接市价吃单」：这能有效减少两笔单子之间的滑点，降低敞口风险。
    7. 价格类型选择「市价」，点击「双腿下单」搞定。

    特别注意：反向套利（费率为负的情况） 当费率为负，我们需要做多合约收息。正确对冲姿势是：现货杠杆借币卖出（做空现货）+ 永续合约做多。但借币会产生利息，如果资金费率收益盖不住借币利息，这单就不划算，新手建议优先做正费率套利。
