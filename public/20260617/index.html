    币安美股交易怎么操作？ #

    币安前几天上线了一个挺重磅的功能——直接交易真实美股，不是那种代币化的美股，是通过持牌券商清算结算的真实股票。我第一时间去试了一下，整体感受是：对于想买美股但嫌开海外券商麻烦的人来说，这个入口确实把门槛拉低了很多。下面我把整个流程从头到尾跑一遍，你照着做就行。

    币安美股是怎么实现的？资产安全吗？ #

    很多人第一个疑问就是：币安又不是券商，它怎么让我买美股？这里简单解释一下。

    我们在币安下单的时候，币安只是一个前端入口，实际的订单会被派发给阿联酋一家持牌券商叫做 Next Trading，再由 Next Trading 把订单转给美国持牌券商，由美国券商负责清算、结算、交割和托管。也就是说，你的资产是由美国持牌券商托管的，不是由币安托管，而且受美国投资者保障机制保护，整个流程是合规的。

    出入金这块也比传统路子方便很多。以前买美股要开海外银行，在银行和券商之间来回汇款，麻烦得很。现在用币安，你可以直接用微信、支付宝买加密货币，然后拿这个去交易美股。如果你有海外银行账户，走美元公司户直接出入金也没问题。

    注册币安账号要怎么做？ #

    你可以点击下方币安交易所的注册链接，复制到浏览器打开。

    币安：https://www.bsmkweb.cc/zh-CN/join?ref=BTC147

    注册时优先用邮箱，推荐 Gmail 或 Outlook，填好邮箱后点「继续」。 币安会往你邮箱发一封验证码邮件，回去查收，把验证码填进来。 设置密码，要求8位以上、至少含一个数字和一个大写字母，设好后点「继续」。 注册成功后它会问你有没有邀请人，选「有」，然后在邀请ID栏填入BTC147。用这个邀请码注册，后续所有交易手续费永久享受8折优惠。如果你是直接从我的注册链接进来的，这里应该已经自动填好了，没有的话手动补上，长期下来能省不少。

    苹果手机怎么下载币安APP？ #

    安卓用户直接去官网下载APK就行，麻烦的是苹果用户。大陆区 App Store 搜不到币安，需要切换到台湾区 Apple ID 才能下载，注意千万不要用美区，美区的版本我们用不了。

    在浏览器搜「台湾苹果ID购买」，找一个卖家购买一个台湾区账号。 进入手机的 App Store，把大陆 Apple ID 退出登录，登录刚买的台湾 Apple ID。 搜索「币安」或「Binance」，找到后下载安装。 下载完成后打开 APP，用刚才注册好的账号登录进去。

    KYC身份认证怎么做？ #

    登录后币安会直接跳到身份认证页面，这一步是必须完成的，不做 KYC 就没办法正常使用交易功能。

    居住国家/地区选中国，证件签发国家/地区同样选中国。 证件类型选身份证，拍摄或上传身份证正反面。 完成人脸识别。 确认个人信息，回答工作状态的问题（如实选就行）。 提交审核，通常几分钟内就能通过，整个流程很顺畅。

    我第一次做 KYC 的时候以为会很慢，结果五分钟不到就过了，比某些传统券商开户快多了。

    用人民币买USDT，C2C怎么操作？ #

    KYC 过了之后就可以入金了。我们要买的是USDT 或 USDC，这两个都是美元稳定币，1:1挂钩美元。如果你是为了交易加密货币，买 USDT，流动性更好；如果目的是交易美股，买 USDC，手续费更便宜。下面以 USDT 为例演示。

    在币安首页点击「添加资金」，选择「C2C买币」。 在自选区选择商家，先筛选好要买的币种（USDT）和金额，比如1000人民币。 支付方式可以选微信、支付宝、银行卡或QQ钱包，我个人习惯用支付宝，专门注册了一个账号用来出入金，隔离风险。 建议勾选「神盾」商家——这类商家在平台压了大量保证金，如果交易出问题导致你账户被封控，平台会赔付一定金额，更有保障。 从筛选出来的商家里找一个成交量大、成交率高的，点击「买入」。 再次输入支付金额或购买数量，点「下单」。

    下单之后有几个注意事项，新手尤其要看清楚：

    点了下单就尽量把这笔交易完成，前几笔交易的完成率会影响你后续 C2C 的畅通度。 点「聊天」看商家的要求，通常是实名转账、不要写任何备注、不要设置延迟到账。 去支付宝给商家转账，备注栏什么都不要写，更不能涉及任何跟币相关的字眼。 如果支付宝转账时遇到系统风控，不要继续尝试，直接取消订单，换一个商家。

    转账完成后回到币安，点「上传付款证明」，上传截图，勾选「我已完成转账」，再点「我已完成付款通知卖家」。卖家收到钱之后会放币，通常一两分钟搞定。

    币安APP里找不到美股入口怎么办？ #

    这是很多人会卡住的地方——买完币之后打开 APP，怎么都找不到美股功能。其实只需要把语言切换成繁体中文就能解锁。

    点击左上角的三条横杠（菜单按钮）。 点右上角的「设置」。 找到「语言」选项，在搜索框里输入「中」，找到「繁体中文」点击切换。 切换成功后，点底部中间的「交易」按钮，上方就会出现「股票」模块，里面有美股和ETF。

    币安里有哪些美股和ETF可以交易？ #

    目前币安里有超过7000只美股和ETF，基本把市场主流标的都覆盖了。

    个股方面，英伟达、特斯拉、苹果、微软、谷歌这些热门的都有；ETF 方面，标普500的 SPY、纳斯达克的 QQQ、半导体指数 SOXX 这些也都在里面，选择空间很大。

    怎么在币安里买入一只美股？以英伟达为例 #

    在股票模块的搜索栏直接搜索「英伟达」，点开第一个结果。 点右上角的小烛台图标可以查看K线图和基本面数据，我对比过币安和富途牛牛的显示，数据基本一致。 在买入区域，支付币种优先选 USDC，手续费比用 USDT 便宜。如果你账户里只有 USDT，可以在现货交易里先把 USDT 换成 USDC。 选择订单类型：限价单是你设定一个价格，等市场到价才成交；市价单是按当前最优价立刻成交。如果用市价单，建议在盘中时段操作。 我这里选限价，输入想要的买入价格（比如200），再输入购买数量——这里支持碎股交易，买0.1股也完全没问题。 点「预览」，确认无误后点「买入」，订单就挂出去了。想取消的话点「撤销」就行。

    币安美股的手续费贵不贵？和其他券商怎么比？ #

    币安本身不收每股佣金，而是收交易价差。具体费率是这样的：

    正常费率：1/1000（即0.1%） 6月活动期间打对折：5/10000（即0.05%） 每笔订单最低收 0.17美元，适用于350美元及以下的订单 碎股交易收 5/10000 账户开立、维护、日常使用全部免费 USDC 兑换美元完全免费；使用其他加密货币会涉及市价兑换价差，所以优先用 USDC 股息处理费0美元；美国税款预扣税率为股息的30%，这个和一般券商一样 监管费用由币安平台承担，用户免费——这点比一些香港和美国券商要划算

    我用实际交易金额来对比一下，感受更直观：

    买10U的英伟达：手续费 0.17U 买2000U的英伟达：手续费 1U（和香港复兴证券单笔最低费用一样） 买3000U的英伟达：手续费 1.5U（比富途牛牛和老虎证券还要便宜一些）

    币安里怎么做空美股？用合约操作 #

    如果你在币安股票板块点「卖出」想做空，会发现它提示没有股票可借——币安目前没有融资融券功能，没办法像传统券商那样直接融券做空。

    但不是没有办法。币安作为加密货币交易所，里面有美股代币的合约，可以通过合约来做空美股。我来演示一下整个流程。

    在币安 APP 底部找到「合约」板块，点进去。 选择「U本位合约」，在分类里找到 TradFi（传统金融），美股就在这里面，另外还有黄金、白银、原油、大豆等商品。注意这里不是所有美股都有，只有部分热门标的。 搜索「英伟达」，点进英伟达的合约交易页面。 合约账户里需要有 USDT 作为保证金，确保账户里有足够的 USDT。

    在开仓之前，先理解一个合约独有的概念——资金费率：

    资金费率为正数时，做多方向做空方支付费用； 资金费率为负数时，做空方向做多方支付费用。 它的作用是让合约价格持续贴近现货真实价格，持仓时需要留意这个费用。

    开仓步骤： #

    开仓模式选「逐仓」，这样每个仓位的风险是独立隔离的，不会让一笔亏损影响其他仓位。 杠杆倍数建议开3到5倍，不要贪高，杠杆越高风险越大。我这里选5倍举例，投入100U本金，实际可以开500U的仓位。 订单类型选市价，可以快速成交。 开仓前一定要设置止盈和止损——我的习惯是止损设在20%，止盈可以设得高一些，先保护好本金。 确认好之后点击「卖出/做空」，做空仓位立刻开好。 页面会实时显示你的盈亏状态。想平仓时点「全部平仓」→「市价全平」→「确认」，立刻平掉。

    做多的操作和做空一样，只是点「买入/做多」而已。

    常见问题 #

    Q：用币安买美股，资产会不会有风险？ #

    实际持仓的美股是由美国持牌券商托管的，不在币安手里，受美国投资者保障机制保护，整体是合规的。当然任何平台都有运营风险，建议不要把所有资产集中在一个地方。

    Q：C2C买币真的不会被冻卡吗？ #

    我个人用下来觉得还好，选神盾商家、按商家要求实名转账、备注什么都不写，这几点做到位，冻卡概率很低。遇到支付宝风控就换商家，不要硬来。另外建议用一个专门的支付宝账号来做出入金，和日常账号隔离。

    Q：合约做空和直接融券做空有什么区别？ #

    融券做空是借来真实股票卖出，合约做空是纯粹押注价格方向，两者没有关系。合约可以加杠杆，但也会产生资金费率，持仓时间越长需要注意费用累积。如果只是想短线博一把方向，合约够用；如果想长期持有空单，传统融券会更合适。
