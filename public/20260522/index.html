    核心概念扫盲 #

    在开始实操前，我们先理清几个新手最常见的疑惑。

    1. 买的币存在哪里？ 在传统金融中，钱存在银行，银行和系统都知道你的余额。但在推崇去中心化的加密货币世界里，币是存在区块链上的“钱包”中的。只要知道钱包地址，任何人都能在区块链浏览器上查到该地址的余额，这就是区块链的透明性。但在交易所（如欧易）中买币，币暂时存放在交易所为你分配的账户里，类似于银行模式。

    2. 在哪里可以买币？ 由于政策原因，目前中国大陆没有统一的平台支持直接用人民币购买加密货币。因此，我们需要通过交易所的 C2C（用户对用户） 板块进行交易。交易所充当类似“支付宝”的担保角色：你付款给商家，商家给你放币，平台保障双方资金安全。

    3. 什么是USDT稳定币？ USDT（泰达币）是一种价格与美元1:1锚定的加密货币。因为它的价格不像比特币那样剧烈波动，所以被称为“稳定币”。新手入圈，通常的第一步就是用人民币购买USDT，然后再用USDT去兑换其他加密货币。

    为了能够使用人民币购买加密货币，你必须完成身份认证。登录APP后，点击左上角进入「个人资料认证」页面：

    • 基础认证： 仅需提供姓名和身份证号码。

    第二阶段：买币与交易操作 #

    1. 怎么用人民币买USDT？ #

    认证通过后，回到首页即可开始购买新手必备的USDT。

    • 点击首页的「我要买币」。
    • 选择「快捷买币」（系统自动匹配优质商家，操作更简单）。

    • 选择购买币种为 USDT。
    • 输入你想购买的人民币金额（例如：100元），系统会自动估算能买到的USDT数量。
    • 选择支付方式（支持支付宝、微信支付或银行卡），点击下一步。
    • 按照页面显示的卖家收款信息进行转账。
    • 转账成功后，务必点击「已完成/我已付款」，等待商家放币即可。

    2. 资金划转：为什么可用资金为0？ #

    当你准备用买到的USDT去交易比特币时，可能会发现交易界面的可用余额显示为0。这是因为刚买到的USDT默认存放在「资金账户」，你需要将其划转到「交易账户」。

    • 在交易界面的金额栏旁，点击蓝色的划转箭头。
    • 选择划转币种为 USDT。
    • 输入划转数量（或点击全部），点击确认。划转瞬间完成且无手续费。

    3. 如何购买其他加密货币（如比特币）？ #

    有了交易账户中的USDT，你就可以在现货市场自由买卖了。

    • 点击APP底部的「交易」。
    • 在屏幕上方点击交易对名称（如默认的 ETH/USDT），搜索并选择你想买的币种，例如 BTC/USDT（用USDT买卖比特币）。

    • 选择你的委托类型，输入金额后点击「买入」。卖出时同理，将滑块拉至想要卖出的比例，点击「卖出」即可换回USDT。
    委托类型 核心区别 适用场景
    市价委托 以当前市场最优价格立即成交。 简单快捷，适合觉得现价合适、想要立刻买卖的新手。
    限价委托 设定一个目标价格，等市场达到该价格时自动成交。 预期价格会下跌想要抄底，或预期会上涨提前挂单。

    第三阶段：WEB3钱包进阶使用 #

    欧易APP内置了自研的WEB3链上钱包（去中心化钱包），入口在APP首页的最上方。

    1. 创建与备份WEB3钱包 #

    • 在APP顶部切换至「WEB3钱包」。
    • 点击左上角或右上角的「钱包管理」，选择「添加钱包」。
    • 选择「创建新的钱包」，系统会提供无私钥和助记词两种模式，建议选择安全性更高的助记词钱包。
    • 创建成功后，点击「立即体验」 -> 「备份钱包」 -> 「抄写备份」。
    • 输入你设置的密码，将屏幕上显示的12个英文单词（助记词）按顺序准确抄写在物理纸张上。
    • 抄写完成后点击「已记录」，按顺序点选助记词完成验证即可。

    安全红线： 助记词就是你钱包的“最高控制权”。绝对不能泄露给任何人，不要截图保存在手机相册或云盘中。 如果助记词丢失，没有任何客服能帮你找回资产！

    2. 交易所与WEB3钱包的资金互转 #

    从欧易交易所转账到WEB3钱包（提币）：

    • 在WEB3钱包首页点击「接收」。
    • 选择「从交易所账户快捷提币」。
    • 选择你要转出的币种（如 USDT）。
    • 选择转账网络（例如推荐新手使用的 ARB-one 网络，手续费较低）。网络就像不同的转账通道，必须确保发送方和接收方使用的是同一条网络。
    • 系统会自动跳转到交易所提币页面并填好地址，输入提取数量，确认手续费后点击「提交」即可。

    从WEB3钱包转回欧易交易所（充值）：

    • 在WEB3钱包首页点击「发送」。
    • 选择币种（如 USDT）。
    • 在接收地址处，直接选择「交易所账户」（系统会自动关联你的欧易账户）。
    • 点击下一步，输入转出数量。
    • 确认转账信息和网络矿工费（Gas费）无误后，点击「确定」即可快速充值回交易所。

    常见问题解答 (FAQ) #

    Q：为什么我的USDT转账一直没到账？ A：请排查两点：第一，确认转账网络是否一致（发送端和接收端必须选择完全相同的网络）；第二，区块链网络偶尔会拥堵，可复制交易哈希值（TxID）在对应的区块链浏览器中查询确认状态。

    Q：WEB3钱包的助记词忘记了或弄丢了怎么办？ A：如果你的设备已经卸载了钱包且没有备份助记词，资产将永久无法找回。去中心化钱包没有中心化服务器保存你的数据，欧易客服也无法帮你重置。请务必在创建的第一时间用纸笔妥善备份。

    Q：C2C买币时，钱转给商家了但一直没收到币怎么办？ A：首先确认你是否在付款后点击了「已付款/已完成」按钮。如果已点击且超过30分钟商家仍未放币，可以直接在订单页面联系欧易客服申请仲裁。平台会对商家收取保证金，能够有效保护买家的正当权益。

    Q：第一次使用钱包转账有什么建议？ A：无论转账给谁，第一次务必先用极小额资金进行测试。确认小额顺利到账后，再进行大额转账，这是避免因操作失误导致资金全损的最有效方法。
