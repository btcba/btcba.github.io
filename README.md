许多新手对虚拟货币的合法性充满疑问。以下为简要解答：

1. **在中国开设虚拟货币企业是违法的**
   根据我国现行法律，在境内设立虚拟货币交易所或从事相关企业经营属于违法行为。

2. **个人购买或持有虚拟货币不违法**
   法律并未明确禁止个人购买或持有虚拟货币，因此在不违反其他法律法规的前提下，个人行为属于合法范围。

自2024年以来，国内未出台新的针对虚拟货币的法律，但牛市期间监管部门可能会发布风险提示性公告。例如 2013 年的比特币风险通知和 2017 年“9·4 通知”，多为市场规范指导文件。
